# Image Steganography Project

Энэ төсөл нь зурагт нууц мессеж/файлуудыг дараах байдлаар нууцлана:
- LSB
- AES encryption
- DCT compression-resistant method

## Features
- Hide text
- Hide ZIP files
- AES encryption
- Password protection

## Technologies
- Python
- OpenCV
- Pillow
- NumPy
- Cryptography

## Requirements

All dependencies are listed in `requirements.txt`:
- flask
- pillow
- pycryptodome
- opencv-python
- numpy

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
